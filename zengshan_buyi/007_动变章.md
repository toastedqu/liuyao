# 《增删卜易》007章、动变章

---

六爻静则不变,动则必变。阳动变阴，〇为阳，动则变━ ━；阴动变阳，╳为阴，动则变━━━。

|   |   |
| --- | --- |
| 假如乾卦初爻动，则变为巽卦： | 假如乾卦上下爻动，则变为坎卦： |
|   |   |
| 假如坤卦三爻动，则变为乾卦： | 假如坤卦中爻动，则变为坎卦： |
|   |   |

假令占得“泽天夬”卦变出“天风姤”卦。

左“泽天夬”为正卦，所变“天风姤”为之卦。以姤卦初爻之丑土，写在夬卦初爻发动子水之傍，谓之子水变出丑土；以姤卦第六爻之戌土，写在夬卦六爻发动未土之傍，即是未出变出戌土。余爻不动不变，不必写出。

安六亲者，姤卦之丑土戌土原是父母，今俱写兄弟者，盖六亲须照前卦之生克而安也，前卦泽天夬，土为兄弟。所以变卦之丑戌二土，亦写兄弟。余卦仿此。

**[乾按]**爻动卦必变，变卦中六亲须按正卦而推。初学者往往照图而写，则大错特错，断卦必难准确。万望慎而又慎。
