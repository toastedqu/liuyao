# 《增删卜易》084章、买何货为吉章

---

野鹤曰：古以金财而像珠宝玉石，水财以类鱼盐，火财陶冶果实，木财竹木，土财五谷。青龙木利，又兼喜庆之用；白虎金财，丧仪屠宰之物；玄武水利；朱雀火宜；螣蛇利于出入；勾陈利于农工。

余以为此乃揣摩五行之论，非经验者也，屡试不验，间有验者，亦偶尔凑合耳，不可为法，必须意属何类，指其事而占之，方能有效。

如：未月 戊申日 (旬空：寅卯)，占往东粤探亲，可带何货归来得利？得“火山旅”

```
                                    离宫：火山旅(六合)
                  朱雀　　　　　　　兄弟己巳火 ━━━
                  青龙　　　　　　　子孙己未土 ━　━
                  玄武　　　　　　　妻财己酉金 ━━━ 应
                  白虎　官鬼己亥水　妻财丙申金 ━━━
                  螣蛇　　　　　　　兄弟丙午火 ━　━
                  勾陈　父母己卯木　子孙丙辰土 ━　━ 世
```

断曰：爻逢六合，世应相生，此行遂意。如问应买何物，余无此法，不敢相误，但喜既得吉卦，任尔冬月卖扇，夏天卖毡，亦得厚利。后此人买檀香翠羽而归，俱获大利。若执古法而断者，则此卦酉金为财，可买珠宝玉石耶？财临玄武，可买水利鱼盐耶？

**［乾按］** 此类占断不可拘守，只须指其事而占之，应方财旺，卦逢六合，世应相生，进货必得厚利。
