# 《增删卜易》074章、功名到何品级章

---

**水一、火二、木三、金四、土五，官临火者，官居二品，官星属水，一品之尊。**

野鹤曰：右古法也。但水止一数，位至公侯者，何以定之？官设九品，数止于五、六、七品者，何以定之？余有指占之法在，我能公侯将相否？我能一品八座否？旺官临日月，指日可期；官破及刑冲，终身失望；子孙持世，犹如止渴望梅；官动生身，恰似探囊取物。

如：巳月 乙卯日（旬空：子丑）曾有邑宰，问将来官至方面否？占得雷山小过卦：

```
                                     兑宫：雷山小过(游魂)
                   玄武　　　　　　　父母庚戌土 ━　━
                   白虎　　　　　　　兄弟庚申金 ━　━
                   螣蛇　子孙丁亥水　官鬼庚午火 ━━━ 世
                   勾陈　　　　　　　兄弟丙申金 ━━━
                   朱雀　妻财丁卯木　官鬼丙午火 ━　━
                   青龙　　　　　　　父母丙辰土 ━　━ 应
```

午火官星持世，后果官至佥事。

又有子月乙亥日，官至方伯，问将来能于开府否？连占三卦，子孙持世，后果降调而归。又因占会试，许之必中，又问：能鼎甲否？余曰：须再占一卦：

于：卯月 甲申日 (旬空：午未)，得“节之蹇”

```
                                     坎宫：水泽节(六合)　　　　兑宫：水山蹇
                   玄武　　　　　　　兄弟戊子水 ━　━ 　　　　兄弟戊子水 ━　━
                   白虎　　　　　　　官鬼戊戌土 ━━━ 　　　　官鬼戊戌土 ━━━
                   螣蛇　　　　　　　父母戊申金 ━　━ 应　　　父母戊申金 ━　━ 世
                   勾陈　　　　　　　官鬼丁丑土 ━　━ 　×→　父母丙申金 ━━━
                   朱雀　　　　　　　子孙丁卯木 ━━━ 　○→　妻财丙午火 ━　━
                   青龙　　　　　　　妻财丁巳火 ━━━ 世○→　官鬼丙辰土 ━　━ 应
```

断曰：古以蹇为不吉，余惟重用神。此卦世临巳火，卯月生之，申日合之。青龙持世，中鼎甲而无疑。但非今科之鼎甲也。公曰：何也？余曰：世爻变出辰土之官，乃辰年之鼎甲也。公曰：此数暗含我机。我少年得一预兆，亦应辰年。场毕即回，至甲辰科殿试，果得首唱传胪。所以此教人指其事而占者，无不验也。何必谆谆执古法而猜之。

**[提要]**本章所论，野鹤之言极是。然古法也有可参照之处，学者可验之而后定论。
