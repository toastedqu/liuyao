# 《增删卜易》073章、僧录、道纪、医官、杂职、阴阳等官章

---

**官医官，亦喜官文发动。**旧文：岂可文书发动。谓僧道医官之占功名，以子孙为用神，不宜父动而克子也？余以为非也。若他人占僧道医人，则以子孙为用神；子孙旺相，道业高强；父母兴隆，庸愚懦弱。若僧道医家自占道术，亦以子孙为用神，子孙旺相，鬼服魔降。

今自占功名，子动则克官矣，如何反为用耶？当仍看官爻。官星持世，或日月动爻以相生；父母持世，或得旺官而相生，其名必成。

**阴阳杂职，俱以子动为嫌。**

子孙动、子孙持世，皆许必不成名，余则以俗家求名同断。

如：丑月 丙辰日 (旬空：子丑)，僧官因官事，恐革退僧官而卜之，得“屯之既济”

```
                                   坎宫：水雷屯　　　　　　　坎宫：水火既济
                 青龙　　　　　　　兄弟戊子水 ━　━ 　　　　兄弟戊子水 ━　━ 应
                 玄武　　　　　　　官鬼戊戌土 ━━━ 应　　　官鬼戊戌土 ━━━
                 白虎　　　　　　　父母戊申金 ━　━ 　　　　父母戊申金 ━　━
                 螣蛇　妻财戊午火　官鬼庚辰土 ━　━ 　×→　兄弟己亥水 ━━━ 世
                 勾陈　　　　　　　子孙庚寅木 ━　━ 世　　　官鬼己丑土 ━　━
                 朱雀　　　　　　　兄弟庚子水 ━━━ 　　　　子孙己卯木 ━━━
```

子孙持世，克官之神，官又临于应爻，此官已属他人矣。后果退职。

**[提要]** 此章所论，亦宜区分所占何人，所占何事，不然难取用神，用神取错则全盘皆错。篇末野鹤之论甚详，宜细参阅。
