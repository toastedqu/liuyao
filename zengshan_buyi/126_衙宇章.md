# 《增删卜易》126章、衙宇章

---

野鹤曰：凡占旧衙署者，与占旧家宅者同断。有因人口多病，有因连任伤官，有因久不得升迁，有因鬼祟现形，皆可指其所疑者而卜。

曾因衙署，连任不利于官长，若非怪刑，即见凶亡，疑官衙有不利。迎余于府中，卜过半月，但有所疑，俱曾占过，皆不现爻，又卜后沟从后流宜否。

于：戌月 己亥日 (旬空：辰巳)，占得“火风鼎”

```
                                    离宫：火风鼎
                  勾陈　　　　　　　兄弟己巳火 ━━━
                  朱雀　　　　　　　子孙己未土 ━　━ 应
                  青龙　　　　　　　妻财己酉金 ━━━
                  玄武　　　　　　　妻财辛酉金 ━━━
                  白虎　　　　　　　官鬼辛亥水 ━━━ 世
                  螣蛇　父母己卯木　子孙辛丑土 ━　━
```

余曰：即此处也，速宜改向东流。后开沟，见许多尸骸白骨，余始悟：白虎鬼持世，是以有之。

再请占一卦：修沟之后何如？得“噬嗑之无妄”

```
                                    巽宫：火雷噬嗑　　　　　　巽宫：天雷无妄(六冲)
                  勾陈　　　　　　　子孙己巳火 ━━━ 　　　　妻财壬戌土 ━━━
                  朱雀　　　　　　　妻财己未土 ━　━ 世×→　官鬼壬申金 ━━━
                  青龙　　　　　　　官鬼己酉金 ━━━ 　　　　子孙壬午火 ━━━ 世
                  玄武　　　　　　　妻财庚辰土 ━　━ 　　　　妻财庚辰土 ━　━
                  白虎　　　　　　　兄弟庚寅木 ━　━ 应　　　兄弟庚寅木 ━　━
                  螣蛇　　　　　　　父母庚子水 ━━━ 　　　　父母庚子水 ━━━ 应
```

断曰：未土财化申金官，未年一定高迁。后于申年升任，应申年者，值官之年也。

**［乾按］** 此占与旧宅同法。宜细参卦例中变化之妙，即有得也。
